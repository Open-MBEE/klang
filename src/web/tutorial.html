Introduction

Welcome to K. K is a textual language intended to be used by modelers, programmers, system designers, and specification writers. The motivation for K is based on the need for a language that provides the right set of constructs and tools to model, verify, and implement.

Classes, Propreties, Functions, and Inheritance

Classes are a basic method of encapsulating all relevant information, properties, functions etc. about a single entity. They provide the simplest approach for abstracting a concept or concrete entity and representing it in your model/program. The follwing figure shows an example of classes being used to model shapes.
