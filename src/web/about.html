About K

K is a wide-spectrum formal modeling language intended for modeling as well as programming. It is based in predicate logic and set theory, but adds to this classes and multiple inheritance. K has been developed as textual language for representing SysML models, thereby part of a solution to connect visual and textual modeling. K models can be checked for satisfiability via a translation to SMT-LIB's input language (first order predicate logic), which currently is processed by Z3.
